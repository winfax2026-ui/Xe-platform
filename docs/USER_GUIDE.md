# 股票及虛擬貨幣平台 XE 版 - 用戶指南

## 快速開始

### 1. 註冊帳戶
1. 訪問網站首頁
2. 點擊「註冊」
3. 輸入邀請碼（預設：**XE2024**）
4. 填寫用戶名、電郵和密碼
5. 🎁 註冊即送 **10 流量金** 開通獎勵

### 2. 開始挖礦
1. 前往「⛏️ 挖礦」頁面
2. 點擊「開始挖礦」
3. 每 **10秒** 自動產出 **1 流量金（1 TC）**
4. 流量金自動累積，24小時不間斷

### 3. 貨幣兌換
1. 前往「💱 兌換」頁面
2. 選擇來源貨幣和目標貨幣
3. 輸入金額，系統會自動計算即時匯率
4. 點擊「確認兌換」
5. **1 TC 永久固定 = 1 USD**

### 4. 完成 KYC 認證
提款前必須完成 KYC：
1. 前往「🔐 KYC 認證」
2. **第一步**：發送電郵驗證碼，驗證您的電郵
3. **第二步**：提交身份資料（姓名、證件號碼）
4. 審核通過後即可解鎖提款功能

### 5. 提款
完成 KYC 後：
1. 前往「🏦 提款」
2. 選擇提款貨幣（TC / USD / HKD）
3. 選擇提款方式：
   - **轉數快（FPS）** — 支援香港各大銀行即時到帳
   - **PayPal** — 即時提款
   - **微信支付（香港）**
   - **支付寶（香港）**
   - **銀行轉帳**
4. 輸入金額，系統顯示手續費和預計到帳金額
5. 提交並輸入驗證碼確認

### 環境變數
| 變數 | 說明 | 預設值 |
|------|------|--------|
| SECRET_KEY | Flask 密鑰 | 自動生成 |
